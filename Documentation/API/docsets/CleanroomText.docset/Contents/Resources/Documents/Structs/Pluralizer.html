Pluralizer

public struct Pluralizer

A Pluralizer instance represents a term that can take multiple forms depending on a quantity.

  • singular

    Specifies the form of the term to be used when the quantity is one.

  • plural

    Specifies the form of the term to be used when the quantity is zero. If nil, the value of the plural property will be used when the quantity is zero.

  • init(singular:plural:none:)

    Initializes a Pluralizer instance with the forms of the term it will represent.

    Within the strings specified for the various forms, the text `{#}` will be replaced with the quantity passed to the termWithQuantity() function.

    Declaration

    Swift

    public init(singular: String, plural: String, none: String? = nil)

    Parameters

    singular

    Specifies the singular form of the term, which is used when the applied quantity is one.
    plural

    Specifies the plural form of the term, which is used when the applied quantity is greater than one.
    none

    Specifies the form of the term to be used when the applied quantity is zero. This parameter is optional; if this value is not specified or is nil, the plural form will be used when the quantity is zero.
  • init(allForms:)

    Initializes a Pluralizer that uses the same form for every quantity.
